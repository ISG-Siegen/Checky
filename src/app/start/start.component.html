Welcome to Checky, the Submission Checklist Generator

Checky is a free web-based, open-source submission-checklist generator. The purpose of Checky is twofold. First, Checky facilitates the creation and management of checklists with a checklist archive, a checklist generator, and a recommender system for checklist items. Second, with Checky we want to raise awareness of the value of submission checklists and initiate discussions about if and how the recommender system community should utilize such lists.
