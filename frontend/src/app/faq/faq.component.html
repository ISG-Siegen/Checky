Frequently Asked Questions (FAQ)

Welcome to the Checky FAQ section! Here, we answer common questions about the platform and its features.

Checky is a free, open-source submission checklist generator that helps researchers create and manage checklists for paper submissions.

To create a checklist, navigate to the Generator, enter a name, search for existing checklist items, and add questions manually or via the recommender system.

Yes! You can save your checklist for later editing, share it with colleagues, or export it as LaTeX for academic use. You can access your saved Checklist via the provided link.

If you need help or want to report a bug, visit our GitHub repository or contact the developers directly via the provided contact information.

The Checky Archive contains a collection of existing checklists from conferences and journals in the field of (automated) machine learning and recommender systems. Each checklist is available as a PDF and/or LaTeX file.

Checky integrates the ChatGPT API to suggest additional questions for your checklist. Based on the items you have already added, ChatGPT generates relevant suggestions to complete your checklist.
